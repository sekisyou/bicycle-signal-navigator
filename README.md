# Bicycle Signal Navigator

自転車利用者向け信号予測ナビアプリ

## 背景・目的
大学まで片道27kmを自転車で通学する中で、
信号待ちによる時間ロスやストレスを感じたことをきっかけに開発しました。

## 主な機能
- ルート検索
- 信号通過予測
- GPS記録
- 走行ログ保存

## 使用技術
- React / JavaScript
- HTML / CSS
- Leaflet / OpenStreetMap

## 工夫した点
- 実走データを用いた予測精度改善
- UIの最適化
- 計算ロジックの改善

## 苦労した点・改善
- GPS誤差への対応
- 予測精度の安定化

## 今後の展望
- AIによる精度向上
- EVナビへの応用
- 実証実験の実施

## 成果
- 実走実験により信号待ち時間の短縮を確認
- 約20km区間で動作検証を実施

## 使い方
1. アプリ起動
2. 目的地入力
3. ルート検索
4. 走行開始
